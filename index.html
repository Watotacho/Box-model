CSS BOX MODEL

Box Model is essentially a box that wraps around every HTML element.
also refers to how HTML elements are modeled in browser engines
and how dimension of those HTML elements are derived from CSS properties.
it a fundamental concept for the composition of HTML webpages

main properties of box model.
1.padding edge-clears an area around the content.
2.border edge-it goes around the padding and content.
3.margin edge-clears an area outside the border
4.content edge-this is where text and images appear
5.width

Total width=width+left padding+right padding+left border+right border+left margin+right margin

Total width=
400px+20px*2+20+*2+2px*2
total=484px.
